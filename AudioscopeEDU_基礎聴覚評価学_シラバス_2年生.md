# 基礎聴覚評価学 シラバス

**Audioscope EDUを活用した聴覚検査教育**

---

## 1. 科目概要

**科目名**：基礎聴覚評価学  
**科目コード**：[科目コード]  
**開講学期**：[前期/後期]  
**対象学年**：2年生  
**対象者**：初学者（聴覚検査の基礎から学ぶ学生）  
**単位数**：2単位（講義1単位、実習1単位）  
**授業回数**：8コマ（各90分、計12時間）  
**授業形態**：講義・実習（Audioscope EDUを活用した実践的学習）  
**使用ツール**：Audioscope EDU（Webベース聴覚検査シミュレーション）  
**関連科目**：聴覚系解剖学、聴覚生理学、臨床聴覚学、言語聴覚障害学概論  
**履修条件**：特になし（初学者向けのため、基礎から学習する）

---

## 2. 授業の目的

本授業では、Audioscope EDUを活用して聴覚検査の基礎を体系的に学習します。実機を使わずに、Webブラウザ上で何度でも練習できる環境を提供し、特に抽象的で理解が難しいマスキングの概念を可視化により理解を深めます。初学者でも段階的に学習できるよう、基礎から応用まで体系的に構成しています。

---

## 3. 到達目標

1. 聴覚器官の解剖学的構造と音の伝導経路を理解できる
2. 聴覚検査の基本概念と意義を理解できる
3. オーディオグラムの読み方と作成方法を理解できる
4. マスキングの概念と必要性を理解し、適切に実施できる
5. タインパノグラム、ART、DPOAEの基本を理解できる
6. 各種検査結果を統合的に解釈できる
7. Audioscope EDUを活用して自立的に学習できる

## 4. 授業計画（8コマ）

### 第1コマ：聴覚器官の解剖と聴覚検査の基礎

**学習目標**

- 音の伝導経路（気導と骨導）を説明できる
- 伝音系と感音系の違いを説明できる
- 聴覚検査の意義と目的を理解できる
- Audioscope EDUの基本操作ができる

**学習内容**

- 音の伝導経路（気導と骨導）
- 伝音系と感音系の違い
- 聴覚検査の意義と目的
- Audioscope EDUの基本操作

**授業の流れ**
1. 講義：聴覚器官の解剖学（35分）

2. 講義：聴覚検査の基礎（20分）
   - 聴覚検査の意義と目的
   - 純音聴力検査、語音聴力検査の概要
   - 聴覚検査の臨床的意義
3. Audioscope EDUデモンストレーション（20分）
   - Audioscope EDUの基本画面の説明
   - 症例の選択方法
   - 各種検査画面の見方

4. 実習：Audioscope EDUでの基本操作練習（10分）

   - オーディオグラムの見方の練習
5. まとめと質疑応答（5分）

**事後学習**：

- Audioscope EDUで正常聴力の症例を3つ以上観察し、特徴をまとめる

## 5. 評価方法

| 評価項目 | 配点 | 評価方法 | 評価基準 |
|---------|------|---------|---------|
| 授業への参加態度 | 20点 | 各コマでの積極的な参加、質問・発言、グループワークへの貢献 | 各コマでの出席、発言回数、質問の質、グループワークでの貢献度を総合的に評価 |
| 事前・事後学習 | 20点 | Audioscope EDUでの学習記録、提出物（各コマの事後学習レポート） | Audioscope EDUでの学習時間、症例演習の実施回数、提出物の内容を評価 |
| 中間課題 | 30点 | 第4コマ終了後に実施（マスキングの理解度確認） | マスキングの概念理解、実施方法、計算能力を評価（記述式・実技形式） |
| 期末課題 | 30点 | 第8コマ終了後に実施（総合的な症例解釈） | オーディオグラム、タインパノグラム、ART、DPOAEの統合的解釈能力を評価（症例問題形式） |
| **合計** | **100点** | | |

**成績評価基準**
- **S（90点以上）**：到達目標を完全に達成し、応用的な問題にも対応できる
- **A（80-89点）**：到達目標を十分に達成している
- **B（70-79点）**：到達目標を概ね達成している
- **C（60-69点）**：到達目標の最低限を達成している（合格）
- **F（60点未満）**：到達目標を達成していない（不合格）

**合格基準**：60点以上

**評価の詳細**
- **授業への参加態度（20点）**
  - 出席：各コマの出席状況（欠席1回につき-3点、3回以上で単位不認定）
  - 発言・質問：積極的な質問・発言（各コマで1回以上推奨）
  - グループワーク：協調性と貢献度

- **事前・事後学習（20点）**
  - Audioscope EDUでの学習記録：各コマの事後学習の実施状況
  - 提出物：各コマの事後学習レポート（8回提出、各2.5点）
  - 学習時間：Audioscope EDUでの総学習時間（目安：各コマあたり1-2時間）

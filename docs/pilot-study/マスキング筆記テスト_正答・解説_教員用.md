## 解説（要点）

### Q1 → B
マスキングは非測定耳が刺激を聞いて応答する「クロスヒアリング」を防ぎ、測定耳の閾値を正確に得るための手法である。

### Q2 → D / Q3 → A
AudioScope EDU および一般的な臨床教育では、AC の IA を 50 dB、BC の IA を 0 dB として扱う。

### Q4 → B
非測定耳の聴力が測定耳より良い（閾値が低い）場合、測定耳への刺激が非測定耳に届きやすく、クロスヒアリングが起こりやすい。

### Q5 → B
オーバーマスキングは、マスキングノイズが測定耳に漏れ、測定耳の閾値が実際より悪く（高く）測定される状態である。

### Q6 → C
気導マスキングは、測定耳の閾値と IA を考慮し、非測定耳に十分なマスキングが届くレベルを設定する。

### Q7 → A
骨導刺激は IA が小さい（0 dB）ため、非測定耳の骨導閾値付近からマスキングを検討する。

### Q8 → B
実効マスキングレベルは、非測定耳の骨導閾値を基準に、マスキングノイズが非測定耳でどの程度有効かを示す概念である。

### Q9 → B
ABG が大きい伝音性難聴では、気導刺激が頭蓋を介して非測定耳に伝わりやすく、マスキングが重要になる。

### Q10 → B
右 60 dB、左 20 dB で右耳を測定する場合、左耳の聴力が良いためクロスヒアリングのリスクがあり、マスキングが必要である。

### Q11 → D
右耳刺激レベルを L dB とすると、IA 50 dB 減衰後に左耳に届くレベルは L − 50 dB。L − 50 = 30 より L = 80 dB。

### Q12 → B
右耳 50 dB 刺激時、左耳には 0 dB 相当が届く。左耳閾値 20 dB より良いため、左耳による偽応答（クロスヒアリング）が起こりうる。

### Q13 → C
AudioScope EDU のオーバーマスキング上限目安は「測定耳 BC + 50 dB」。BC 30 dB のとき上限は 80 dB。

### Q14 → B
マスキング 90 dB は上限 80 dB を 10 dB 超過しており、オーバーマスキングの可能性がある。

### Q15 → B
ABG が大きい場合、気導刺激が対側に伝わる可能性があるため、BC 測定時もマスキングを検討する。

### Q16 → B
両耳の気導閾値が同程度で左右差が小さい場合、片耳 AC 測定では原則マスキングは不要とされることが多い。

### Q17 → B
右耳伝音性・左耳正常に近い場合、右耳 AC 測定時は左耳の気導・骨導閾値と右耳 AC の関係を最優先で評価する。

### Q18 → B
マスキング過大は測定耳へのノイズ漏れを招き、閾値が実際より悪く測定される。

### Q19 → B
適切なマスキングは非測定耳の応答を最小化し、測定耳の閾値決定を支援する（完全保証ではない）。

### Q20 → B
臨床では適切な開始レベルから設定し、クロスヒアリング・オーバーマスキングの兆候を見ながら調整する。

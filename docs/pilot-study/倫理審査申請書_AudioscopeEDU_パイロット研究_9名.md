## 3. 研究の目的

本研究は、[大学名] の聴覚検査関連教育において、Webベース聴覚検査シミュレータ「AudioScope EDU」を**任意の学習協力者（目標9名程度）**に使用してもらい、その**実現可能性（feasibility）**、**受容性（acceptability）**、**予備的教育効果（preliminary learning outcomes）**を評価することを目的とする。

本研究は、将来的な**正式なカリキュラム組み込み**および**大規模評価研究**の前提となる形成的評価（formative evaluation）である。

### 研究課題

1. AudioScope EDU は、聴覚検査（純音聴力検査・マスキング）の学習支援ツールとして、学生に受け入れられるか（満足度・ユーザビリティ）。
2. 使用前後で、マスキングに関する知識および聴覚検査実施に対する自己効力感に変化の傾向があるか。
3. 学習ログ（ログイン回数、症例完了数、症例別正答率）に基づき、小規模でも運用可能か（実現可能性）。
4. 本パイロットの結果を踏まえ、授業への本格導入は妥当か。

### 本研究で主張しないこと

本研究は小規模パイロット（N≈9）であるため、以下は主張対象としない。

- 教育効果の確立
- 従来教育との優越性の証明
- 一般化可能性の確立

## 4. 研究の意義・背景

### 4.1 背景

聴覚検査教育、特にマスキングは、実機・実習機会・指導者の制約により反復学習が不足しやすい。AudioScope EDU は、純音聴力検査（AC/BC）、マスキング可視化、補助検査シミュレーション等をブラウザ上で提供する教育用Webアプリケーションであり、開発は完了している。

### 4.2 現状

現在、[大学名] において、聴覚検査学習を目的とした学生**約9名**が、教員の案内のもと**任意**で AudioScope EDU を試用している。本申請は、この試用を**教育改善の場を超えた研究として正式に実施・評価する**ための倫理審査である。

### 4.3 意義

- 小規模ながら、実運用条件下での受容性・使いやすさを評価できる
- 今後のカリキュラム導入判断の根拠となる
- JMIR Formative Research 等への投稿を想定した形成的評価データを得られる

1. 倫理審査委員会の承認を得る

8. 学習ログを匿名IDと突合し、集計・分析する

### 利益

- 学生：マスキング・聴覚検査の理解促進、反復練習機会

- 成績と研究参加の完全分離
- 匿名ID による仮名化
- カメラ・画面録画は一切行わない
- 同意前データは研究に使用しない

## 13. 統計解析計画（N≈9 に適した計画）

本研究は小規模パイロットのため、**記述統計を中心**とし、推論統計は探索的に実施する。

| 解析内容 | 方法 |
|----------|------|
| 基本特性 | 記述統計（人数、割合） |
| 筆記テスト | T0・T1 の正答率（平均±SD、中央値）、個人別変化量 |
| 自己効力感 | T0・T1 の平均±SD、変化量 |
| SUS | 平均±SD（0〜100点換算） |
| 満足度 | 各項目平均、NPS |
| 学習ログ | ログイン回数、完了症例数、正答率の記述統計 |
| 前後比較 | 対応のある Wilcoxon 符号順位検定（探索的、p値は参考値として報告） |

**注意**: N=9 では検出力が限られるため、有意差の有無よりも**効果の方向・大きさの傾向**を重視する。論文では「preliminary」「feasibility」を明示する。

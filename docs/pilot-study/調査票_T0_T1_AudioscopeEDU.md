- D. オーディオグラムの形状を変えるため

- A. 0 dB
- B. 25 dB
- C. 40 dB
- D. 50 dB

**Q3.** 骨導（BC）刺激におけるインタラウラル減衰（IA）の一般的な値として最も適切なものはどれか。

- A. 0 dB
- B. 25 dB
- C. 40 dB
- D. 50 dB

**Q4.** クロスヒアリングが起こりやすい状況として最も適切なものはどれか。

- A. 両耳の聴力が正常で左右差がない
- B. 測定耳の閾値が非測定耳より良い（閾値が低い）場合
- C. 測定耳の閾値が非測定耳より悪い（閾値が高い）場合
- D. マスキングを十分にかけている場合

**Q5.** オーバーマスキング（過剰マスキング）とはどのような状態か。最も適切なものはどれか。

- A. マスキングが不十分で非測定耳が応答する状態
- B. マスキングノイズが測定耳に漏れ、測定耳の閾値が上がって見える状態
- C. マスキングノイズが小さすぎる状態
- D. 測定耳と非測定耳の聴力が同じ状態

**Q6.** 気導マスキングの開始レベルとして、最も一般的な考え方に近いものはどれか。

- A. 非測定耳の気導閾値
- B. 測定耳の気導閾値

- D. 常に最大出力で行う

**Q7.** 骨導マスキングの開始レベルとして、最も一般的な考え方に近いものはどれか。

- A. 非測定耳の骨導閾値付近から開始する
- B. 測定耳の気導閾値から開始する
- C. IA を考慮する必要はない
- D. 常に 0 dB HL から開始する

**Q8.** 実効マスキングレベル（effective masking level）の説明として最も適切なものはどれか。

- A. オーディオメータのノブ表示値そのもの
- B. 非測定耳の骨導閾値を基準に、マスキングノイズが非測定耳でどの程度聞こえるかを示す値
- C. 測定耳の気導閾値
- D. 両耳の気導閾値の平均

**Q9.** 伝音性難聴で気導と骨導の差（ABG: air-bone gap）が大きい場合、マスキングが特に重要になる主な理由はどれか。

- A. ABG が大きいと必ず感音性難聴になるため

- C. 骨導刺激は常に両耳に聞こえるため
- D. ABG があるとマスキングは不要になるため

**Q10.** 片耳性の感音性難聴（右耳 60 dB、左耳 20 dB）で右耳の気導閾値を測定する場合、マスキングの要否として最も適切なものはどれか。

- A. 不要（どちらの耳も同程度に聞こえる）

3. I thought the system was easy to use.
4. I think that I would need the support of a technical person to be able to use this system.
5. I found the various functions in this system were well integrated.
6. I thought there was too much inconsistency in this system.

**SUS スコア計算**

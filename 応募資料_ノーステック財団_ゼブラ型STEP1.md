## 1. 事業概要

### 1.1 プロダクト名
**HearSim（ヒアシム）**

### 1.2 事業の目的
聴力検査の教育・トレーニングにおける実習機材の不足と高コストの課題を解決し、Webブラウザ上で包括的な聴覚検査シミュレーションを提供するEdTechソリューション。

### 1.3 プロダクトの特徴
- **包括的な聴覚検査シミュレーション**
  - オーディオグラム（聴力図）作成・マスキングトレーニング
  - タインパノグラム（鼓膜可動性検査）
  - アブミ骨筋反射（ART）
  - DPOAE（歪成分耳音響放射）

- **教育に最適化された機能**
  - リアルタイムのマスキング可視化（オーバーマスキング、クロスヒアリング警告）
  - 症例プリセット（A-H、8症例以上）
  - 症例ごとの臨床情報（年齢、性別、主訴、病歴、所見）
  - 解答表示機能による学習効果の向上

- **技術的優位性**
  - ブラウザベース（インストール不要、どこでもアクセス可能）
  - レスポンシブデザイン
  - インタラウラル減衰（IA）やScale-Outロジックなど臨床的に正確なアルゴリズム実装

## 2. 市場分析

### 2.1 市場規模・ニーズ

#### 2.1.1 ターゲット市場

**1. 教育機関（優先市場）**
- **対象**
  - 言語聴覚士養成校（全国約80校）
  - 臨床検査技師養成校（全国約100校）
  - その他（医学部・看護学部の耳鼻咽喉科実習、専門学校・短大）

- **市場規模**
  - 言語聴覚士養成校：約80校、学生数約3,000名/年
  - 臨床検査技師養成校：約100校、学生数約5,500名/年
  - **合計：約180校、年間約8,500名以上の学習者が直接の潜在顧客**
  - **市場規模：年間約180校 × 平均10万円 = 約1,800万円の市場ポテンシャル**

- **課題**
  - 聴力検査機器の高額（1台数百万円）
  - 実習時間・機材の制約
  - マスキングの理解が難しい（概念が抽象的）
  - 実習機会の不足
  - **現在、Webベースの包括的な聴覚検査シミュレーション教材は市場に存在しない（競合なし）**

- **実績**
  - **北海道医療大学**にて既に導入・利用実績あり（パイロット導入成功）

**2. 資格保持者（個人）**
- **対象**
  - 言語聴覚士（約30,000名、年1,000名以上が新規取得）
  - 臨床検査技師（約50,000名、年2,000名以上が新規取得）
  - 認定補聴器技能者

- **ニーズ**
  - 資格試験対策
  - 実務スキル維持・向上
  - 継続教育

**3. 医療機関（耳鼻咽喉科クリニック）**
- **対象**
  - 耳鼻咽喉科クリニック（全国約2,000施設）
  - 総合病院の耳鼻咽喉科

  - スタッフ教育・トレーニング
  - 患者説明ツール
  - 新人教育の効率化

### 2.2 競合分析

**直接的な競合製品は存在しない**
- 現在、Webベースで包括的な聴覚検査シミュレーション（オーディオグラム、タインパノグラム、DPOAE、ART）を提供するソフトウェアは市場に存在しない
- HearSimは**新規市場を開拓するポジション**にある

| 代替手段 | 特徴 | HearSimの優位性 |
|---------|------|----------------|
| 実機（オージオメータ等） | 実際の検査が可能 | **コスト：1/100以下、どこでも利用可能、何度でも練習可能、破損リスクなし** |
| 紙ベースの教材 | 低コストだが静的 | **インタラクティブ、リアルタイムフィードバック、可視化、症例の豊富さ** |
| 既存のシミュレーションソフト | 市場に存在せず | **ブラウザベース、インストール不要、無料から利用可能、継続的改善** |
| YouTube等の動画教材 | 視聴のみ可能 | **実際に操作・練習できる、フィードバック機能、症例のカスタマイズ** |

**競争優位性**
- **ファーストムーバー**：Webベースの包括的な聴覚検査シミュレーションの先駆者
- **実績**：北海道医療大学での実際の導入・利用実績あり
- **技術的優位性**：臨床的に正確なアルゴリズム（IA、Scale-Out等）の実装

### 2.3 市場機会
- **競合不在のブルーオーシャン市場**：直接的な競合製品が存在しない
- コロナ禍でのオンライン教育普及
- EdTech市場の成長（2025年までに年率15%成長予測）
- 医療・福祉教育のデジタル化推進
- 実習機材不足の課題が顕在化
- **実績の裏付け**：北海道医療大学での成功実績により、市場での受け入れが証明済み
- **特許取得予定（可能性が大幅に向上）**：
  - AI症例生成機能の実装により、特許性がさらに高まった
  - 既存のマスキングアルゴリズムに加えて、疾患データベースベースの症例自動生成システムという新たな技術的要素を追加

  - 技術的優位性の保護により、長期的な競争優位性を確保

## 4. 技術・開発

### 4.1 技術スタック
- **フロントエンド**：React.js
- **チャート描画**：Recharts
- **GIF生成**：gif.js
- **データベース**：Supabase
- **ホスティング**：Vercel

### 4.2 主要機能
1. **オーディオグラム作成**
   - AC/BC測定
   - 左右別表示（O/X、</>記号）
   - 標準的な周波数（125Hz-8kHz）

2. **マスキング機能**
   - リアルタイムマスキング可視化
   - オーバーマスキング警告
   - クロスヒアリング警告
   - インタラウラル減衰（IA）ロジック

3. **症例管理**
   - 8症例以上のプリセット（A-H）
   - **AIによる症例自動生成機能（実装済み）**
     - 疾患特異的な症例パターンの自動生成（メニエール病、突発性難聴、耳硬化症、騒音性難聴等8種類以上の疾患パターン）
     - 年齢に応じた適切な臨床情報の自動生成
     - 正常聴力から高度難聴まで、多様な難易度の症例を無限に生成
     - 各症例に整合性のあるタインパノグラム・DPOAE・ARTデータも自動生成
   - 各症例の臨床情報（年齢、性別、主訴、病歴、所見）
   - タインパノグラム・DPOAE・ARTとの連携

4. **GIF出力機能**
   - タインパノグラムGIF
   - DPOAE測定アニメーションGIF
   - 教材作成支援

### 4.3 AI症例生成機能（実装済み）

**既に実装済みのAI症例生成機能により、HearSimは競合製品とは一線を画す価値を提供しています。**

#### 4.3.1 AI症例生成の価値・メリット

**1. 教育的価値**
- **無限の症例パターン**：固定された8症例だけでなく、AIが疾患特異的なパターンを自動生成
  - 正常聴力、感音性難聴、伝音性難聴、混合性難聴の4パターン + 8種類以上の疾患特異的パターン
  - メニエール病、突発性難聴、耳硬化症、騒音性難聴、加齢性難聴、耳小骨離断、音響外傷、ムンプス難聴など
- **反復練習の実現**：同じ症例は二度と出現しないため、真の実力が試される
- **年齢に応じた適切な臨床情報**：年齢、性別、主訴、病歴が疾患特性に基づいて自動生成
  - 例：46歳の場合「職場の健康診断」、18歳の場合「学校検診」など、自然な表現
- **臨床的整合性の確保**：オーディオグラム、タインパノグラム、ART、DPOAEが全て整合性を持って生成

**2. 経済的価値・効率性**
- **教材作成コストの削減**：教員が手作業で症例を作成する必要がない
- **教員の負担軽減**：事前準備なしで多様な症例を使用可能
- **学習機会の拡大**：症例数の制約がなく、学生は何度でも異なる症例で練習可能

**3. 技術的優位性・差別化要因**
- **臨床的精度の高さ**：
  - ISO 7029標準に基づく年齢別正常聴力閾値の反映
  - 疾患特異的なオーディオグラムパターンの再現（C5 dip、Carhart notch、低音障害型等）
  - ティンパノグラム型とオーディオグラムパターンの整合性確保
- **実装済みの即戦力**：既に動作する機能として、すぐに教育現場で利用可能
- **競合製品との明確な差別化**：現在市場に存在する教材では実現できていない機能

**4. スケーラビリティ・拡張性**
- **症例の無限拡張**：新しい疾患パターンや臨床パターンを追加可能
- **学習データの蓄積**：生成された症例と学習者の解答データを蓄積し、将来的なAI機能拡張の基盤となる
- **カスタマイズ性**：教員の要望に応じて、特定の疾患パターンや難易度に特化した症例生成が可能

**5. 学習効果の向上**
- **多様性**：同じ難聴パターンでも、年齢、性別、主訴、病歴が異なるため、実臨床に近い状況を体験
- **難易度の幅**：初心者向けから上級者向けまで、様々な難易度の症例を自動生成
- **統合的理解**：オーディオグラムだけでなく、タインパノグラム、ART、DPOAEも含めた包括的な症例により、総合的な診断思考を養成

#### 4.3.2 AI症例生成の技術的特徴

- **疾患データベース**：8種類以上の疾患の疫学情報、典型的なオーディオグラムパターン、タインパノグラム・ART・DPOAEの特徴を統合
- **パターン認識アルゴリズム**：生成されたオーディオグラムから難聴パターンを自動判定し、最も可能性の高い疾患を推定
- **整合性チェック**：伝音性難聴の場合はタインパノグラムB型/As型/Ad型、感音性難聴の場合はA型など、臨床的に正しい組み合わせを自動生成
- **ISO 7029標準の統合**：年齢別正常聴力閾値を自動計算し、年齢に応じた適切な症例を生成

#### 4.3.3 特許性への寄与

AI症例生成機能の実装により、特許の可能性が大幅に向上しました：

**1. 技術的独自性の強化**
- **統合的な症例生成システム**：オーディオグラム、タインパノグラム、ART、DPOAEの4つの検査結果を臨床的整合性を持って自動生成するシステムは、市場に存在しない
- **疾患データベースベースのアルゴリズム**：疫学情報と臨床パターンを統合した症例生成アルゴリズムは独自性が高い
- **パターン認識と症例生成の組み合わせ**：生成されたオーディオグラムから疾患を推定し、それに基づいて臨床情報を自動生成する技術的組み合わせ

**2. 既存技術との組み合わせによる新規性**
- 既存のマスキングアルゴリズム（インタラウラル減衰、Scale-Outロジック）とAI症例生成の組み合わせ
- ISO 7029標準を統合した年齢別正常聴力閾値の自動計算と症例生成の統合
- Webベースの教育プラットフォームとAI症例生成の統合

**3. 実用性・産業上の利用可能性**
- 教育現場で即座に利用可能な実装済み機能
- 教員の負担軽減、学習効果の向上という明確な効果
- 教材作成コストの削減という経済的効果

**4. 進歩性（非自明性）**
- 単なるランダム生成ではなく、疾患データベースとパターン認識アルゴリズムによる臨床的精度の高い症例生成
- タインパノグラム型とオーディオグラムパターンの整合性を自動的に確保する技術
- 年齢に応じた適切な臨床情報の自動生成

**特許出願の対象技術**
1. **Webベース聴覚検査シミュレーションシステム**（基本特許）
   - マスキングアルゴリズム（インタラウラル減衰、Scale-Outロジック）
   - リアルタイム可視化システム

2. **AI症例自動生成システム**（追加特許または基本特許への追加クレーム）
   - 疾患データベースベースの症例生成アルゴリズム
   - 統合的な症例生成（オーディオグラム+タインパノグラム+ART+DPOAE）
   - パターン認識による疾患推定と臨床情報の自動生成
   - ISO 7029標準統合による年齢別正常値の自動計算

**特許戦略**
- AI症例生成機能は、既存の基本特許（マスキングアルゴリズム）に追加するクレームとして申請可能
- または、独立した特許として申請することで、より広範な技術的保護が可能
- 将来的なAI機能拡張（学習者フィードバック、学習パターン分析）も含めた包括的な特許ポートフォリオの構築が可能

#### 4.3.4 今後のAI機能展開計画

1. **AIによる学習者向け個別フィードバック**（将来実装）
   - 学習者の操作パターン・間違いの傾向を分析
   - 個人の弱点に応じた適応的な学習ガイド
   - マスキング判断の精度向上を支援するAIコーチング

2. **AIによる学習パターン分析**（将来実装）

   - 学習効果の定量的評価

3. **AIによる診断支援機能**（将来展開）
   - オーディオグラムパターンから疾患の可能性を提示（学習支援用）
   - 複数の検査結果の統合分析支援

**技術的実現方法**
- 機械学習モデル：操作ログ・解答データの蓄積と分析
- 自然言語処理：フィードバックテキストの自動生成
